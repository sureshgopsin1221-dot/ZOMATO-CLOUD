# Zomato Clone

A responsive frontend clone of the Zomato website built using **HTML, CSS, and JavaScript**. This project recreates the look and feel of the original website with a clean, modern, and user-friendly interface. It focuses on responsive design, structured layouts, and interactive UI elements to provide a smooth browsing experience across different screen sizes.

## Features

* Responsive design for desktop and mobile devices
* Clean and modern user interface
* Well-structured HTML and CSS
* Interactive elements using JavaScript
* Beginner-friendly frontend project for web development practice

## Technologies Used

* HTML5
* CSS3
* JavaScript
